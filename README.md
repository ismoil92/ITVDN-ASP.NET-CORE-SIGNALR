# ITVDN-ASP.NET-CORE-SIGNALR

### Тема: Практика номер 3. Клиентская часть: JS клиент.

*Задание*
  1. Создайте  клиент-серверную  систему  с  использованием JS  клиента.  Реализуйте возможность  клиентам  обмениваться  между  собой  сообщениями.  Реализуйте  возможность получения текущего времени на сервере за 1 вызов только с клиента и за 1 вызов с клиента, и сервера (синхронный и асинхронный варианты).
  2. Модифицируйте  код  первого  задания,  добавив  возможность  переподключения  на клиенте в 5 попыток, с интервалами 1, 2, 3, 4 и 5 секунд.
  3. Реализуйте собственную политику переподключения на клиенте. Длительность первой попытки должна быть случайной - от 1 до 10 секунд. Все последующие - должны быть в 2 раза больше предыдущей, но общая длительность всего процесса переподключения не должна превышать 1 минуты.
