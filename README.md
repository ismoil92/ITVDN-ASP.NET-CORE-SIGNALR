# ITVDN-ASP.NET-CORE-SIGNALR

### Тема: Практика номер 2. Серверная часть: Hub и HubContext.

*Задание*
  1. Создайте клиент-серверную систему с использованием строго типизированных хабов.Реализуйте логирование событий подключения клиентов. 
  2.  Модифицируйте код первого задания, добавив web api контроллер с методом, который будет оповещать всех клиентов при вызове этого метода.
  3.  Создайте такой хаб, к которому можно подключиться только в чётные минуты времени на сервере.
